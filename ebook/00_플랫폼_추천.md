# 부크크 vs 유페이퍼 — 어디에 내야 하는가

> 결론부터 말씀드립니다. **부크크를 메인, 유페이퍼를 보조**로 **동시 출간** 하는 전략을 추천드립니다. 둘 다 비독점 계약이라 같은 원고를 양쪽에 등록해도 위반이 아니며, 장점이 정확히 달라 서로를 보완합니다.

---

## 왜 이중 출간이 정답인가

| 축 | 부크크 (Bookk) | 유페이퍼 (Upaper) |
|---|---|---|
| 원고 파일 | PDF · Word | PDF(50MB↓) · EPUB |
| 표지 규격 | 템플릿 제공 (가이드에 맞춰 제작) | 가로 600~1000px · 세로 1000px (표지가 원고 파일 첫 장에 포함되어야 함) |
| ISBN | 무료 부여 | 1,000원 · 7일 소요 |
| 전자책 인세 (자사 플랫폼) | 판매가의 **40~60%** | 판매가의 **70%** |
| 제휴사 판매 (교보·예스24·알라딘) | 가능 | 가능 (수수료 40%, 저자 60%) |
| 종이책(POD) | 동시 출간 가능 | 전자책 전용 |
| 가격 제한 | 자유 | 10,000원↑ = 글자 10만자↑, 7,000원↑ = 5만자↑ (※ 본 원고 41.6만자로 19,900원까지 무제한 설정 가능) |
| 기술서 독자 노출 | ◎ (IT·자기계발 판매량 큼) | △ (전자책 특화, IT 카테고리는 상대적으로 얕음) |
| 향후 출판사 역제안 가능성 | ◎ (판매량 누적 시 출판사 쪽에서 먼저 연락) | △ |

본 도서처럼 **416,149자 분량의 기술서**는 두 플랫폼 모두 상위 가격대(19,900원 이상)로 올릴 자격이 되고, 카테고리도 "IT·컴퓨터" 로 명확합니다. 한 곳만 택하면 기회 비용이 큽니다.

---

## 단일 선택이 꼭 필요하다면 — 부크크

다음 세 가지 이유로 단일 선택은 부크크가 유리합니다.

첫째, **종이책(POD) 확장 옵션** 이 그대로 살아 있습니다. 전자책이 유튜브·블로그로 입소문 나기 시작하면 부크크에서 바로 종이책 등록으로 넘길 수 있고, 이때 강의·부트캠프 교재화가 가장 자연스럽게 연결됩니다.

둘째, **IT·기술서 독자 풀** 이 부크크 쪽에 더 두텁습니다. 2024~2025년 부크크 자가출판 IT 카테고리 베스트셀러 중 절반 이상이 개인 저자의 기술 교재입니다. 유페이퍼는 소설·에세이·자기계발이 메인입니다.

셋째, 부크크는 **표지 템플릿·편집 도구** 가 별도 제공되어, 외주 디자이너 없이도 납품 가능한 수준의 결과물이 나옵니다. 유페이퍼는 EPUB 표지의 경우 별도의 이미지 코딩이 필요해 진입 장벽이 조금 더 높습니다.

단점은 **인세율** 입니다. 유페이퍼의 70% 대비 부크크는 40~60% 수준이라 권당 수익이 다소 낮습니다. 이 차이를 유페이퍼 병행 출간으로 보완합니다.

## 권장 출간 순서

1. **W1 · 부크크 1차 등록** — 전자책 + 종이책(POD) 동시 접수. ISBN 자동 부여.
2. **W1 · 유페이퍼 2차 등록** — 같은 원고를 EPUB/PDF 로 올리고 교보·예스24·알라딘 제휴 판매 체크.
3. **W2~ · 유튜브 "All That AI" 런치 시리즈** — 각 장별 해설 영상을 주 2편 공개. 설명란에 부크크/유페이퍼 링크 병기.
4. **W4 · 가격 조정** — 초기 할인(19,900 → 14,900원) 시행해 다운로드 누적.
5. **M2~ · 종이책 전환** — 부크크 판매량이 500부 넘어가면 POD 종이책 가격을 낮춰 대학 교재 채택 영업 병행.

이 경로를 따라가면, 두 플랫폼의 장점을 모두 취하면서도 **하나의 원고·하나의 표지·하나의 책 소개** 로 관리가 가능합니다. 다음 문서들이 그 실행 자료입니다.
