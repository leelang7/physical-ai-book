## 단계 5 · 부크크 업로드

1. [부크크 저자 로그인](https://bookk.co.kr/) → **책 만들기 → 전자책 만들기**
2. 도서 정보 입력 (메타데이터.md 내용 복붙)
3. **원고 파일** 업로드 : `전체원고_v2.0.pdf` (표지 없는 버전)
4. **표지 이미지** 업로드 : `cover_A_red_column.png`
5. 미리보기 재확인 → **ISBN 자동 발급 체크** → **검수 요청**
6. 영업일 3~5일 대기

## 단계 6 · 유페이퍼 업로드

1. [유페이퍼 저자 가입](https://upaper.kr/) → **마이페이지 → 신간 등록**
2. 도서 정보 입력 (같은 메타데이터 재사용)
3. **원고 파일** 업로드 : **표지 포함** PDF
4. 별도 **표지 이미지** 업로드 : `cover_A_red_column.png`
5. 판매 채널 모두 체크 (유페이퍼·교보·예스24·알라딘)
6. **ISBN 1,000원 결제** → 검수 요청
7. 영업일 3~7일 대기
