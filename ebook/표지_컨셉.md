## 신안 A · "Red Column" *(추천)*

[cover_A_red_column.svg](cover/cover_A_red_column.svg)

좌측 60% 딥 블랙(Zinc 900), 우측 40% 레드(Tesla Red) 의 **수직 분할**. 좌측에는 타이포그래피가 제어된 그리드로 자리 잡고, 우측 레드 컬럼에는 세로로 *"ALL THAT AI · VOL.01 · 2026"* 이 회전 배치되며 하단에 거대한 숫자 **01** 이 박힙니다. 제목은 *"테슬라처럼 만드는"* 을 얇은 레귤러로, *"비전 자율주행"* 과 *"피지컬 AI"* 를 거대한 블랙 900으로 받치되 마지막 줄만 **레드 강조**. 기술서의 무게감과 브랜드 시그널이 동시에 붙습니다.

- **컬러** : `#0B0C10` (딥 블랙) · `#E11D2E` (Tesla 레드) · `#F4F4F6` (오프 화이트)
- **인상** : "이 책이 Tesla 스택 이야기구나" 를 0.3초 안에 전달
- **강점** : 썸네일 크기로 축소되어도 레드 컬럼이 눈에 띕니다

## 신안 B · "Swiss Grid"

[cover_B_swiss_grid.svg](cover/cover_B_swiss_grid.svg)

아이보리 배경 + 정갈한 그리드. 상단·중단·하단에 가는 블랙 수평선이 타이포를 **세 구간** 으로 나누고, 우상단에 거대한 오렌지 **01** 이 장식으로 박힙니다. 제목은 네 줄로 좌측 정렬, 마지막 줄 *"피지컬 AI"* 만 오렌지 레드. 하단엔 *FSD v13 · OCCUPANCY · DAgger · OpenVLA · Optimus · Isaac Lab · MentorPi · ROS2* 같은 핵심 키워드 두 줄이 모노스페이스로 들어가, 기술서임을 드러냅니다.

- **컬러** : `#F5F2EC` (아이보리) · `#1A1A1A` (차콜) · `#E94F2E` (오렌지 레드)
- **인상** : 담담하고 지적인 교보문고 매대 스타일
- **강점** : 대학 교재 · 출판사 재제안 시 신뢰감 최상

## 신안 C · "Neon Horizon"

[cover_C_neon_horizon.svg](cover/cover_C_neon_horizon.svg)

네이비에서 딥 블랙으로 흐르는 야간 그라데이션 배경 위에, 가운데 **네온 레드 지평선** 이 가로지르고 지평선 아래로 **원근 그리드** 가 수렴합니다. 지평선 아래 정중앙에 *PHOTON · IN / —— / CONTROL · OUT* 이 얇게 새겨지고, 지평선 위 좌측엔 제목 타이포가 놓입니다. 책의 슬로건을 시각으로 번역한 컷.

- **컬러** : `#060814` (딥 블루블랙) · `#FF3B4D` (네온 레드) · `#FFFFFF`
- **인상** : 기술 컨퍼런스 포스터 느낌. SNS · 유튜브 썸네일 친화
- **강점** : YouTube "All That AI" 채널 썸네일과 통일감 만들기 좋음

## 내부 최종 추천 — A안 (Red Column)

세 안 모두 동일한 **1,600 × 2,400 · 명시 좌표 · 폰트 fallback · 여백 140px** 규칙으로 맞춰져 있어 겹침 없이 렌더링됩니다. 그중 **A안** 을 추천드리는 이유는 다음과 같습니다.

- **썸네일 축소 테스트** 통과 : 128px 섬네일까지 떨어져도 레드 컬럼이 보이고, "테슬라처럼 만드는" 윗줄과 *피지컬 AI* 레드 강조가 구분됩니다.
- **시리즈 확장성** : *"VOL.01"* 표시와 레드 컬럼 하단의 **01** 숫자가 이후 2권·3권에 02·03 만 갈아 끼우면 즉시 시리즈 룩이 유지됩니다.
- **심리적 기호** : 딥 블랙 + 레드는 Tesla·자율주행 카테고리의 시각 관습과 자연스럽게 어울립니다.

B안(Swiss Grid) 은 출판사·대학 채택 채널에, C안(Neon Horizon) 은 SNS·유튜브 브랜드 통일에 강합니다. 판매 전략에 따라 **A안 본판 + C안 SNS 변형** 두 개를 함께 쓰는 것도 좋은 조합입니다.

## SVG → PNG 내보내기

세 가지 방법 중 편한 것을 선택하세요.

- **VSCode** : 각 SVG 파일 열고 우클릭 → *Open Preview* → 브라우저로 넘긴 뒤 Chrome *"다른 이름으로 저장"* 에서 PNG. 가장 빠릅니다.
- **Inkscape** (무료) : 파일 열고 *File → Export PNG Image* · 폭 1600px · 300dpi.
- **Chrome Headless (자동화)** :
  ```
  chrome --headless --screenshot=coverA.png --window-size=1600,2400 cover_A_red_column.svg
  ```
- **Figma** 에 드래그해 놓고 텍스트를 직접 옮기면 한글 자간·커닝을 더 섬세하게 다듬을 수 있습니다.
