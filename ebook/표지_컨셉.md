# 전자책 표지 컨셉 · 디자인 가이드

> 전자책 표지는 상세페이지 썸네일로 가장 먼저 독자 눈에 들어오는 요소입니다. 두 플랫폼의 규격을 모두 만족시키는 단일 표지를 만드는 방법을 정리했습니다.

---

## 최종 규격 결정

두 플랫폼의 요구를 모두 수용하는 **가장 안전한 단일 규격** 은 다음과 같습니다.

| 항목 | 값 |
|---|---|
| 가로 × 세로 | **1,600 × 2,400 px** (비율 2:3, 세로형) |
| 해상도 | 300 dpi (인쇄 대비) |
| 파일 형식 | PNG (배포 시) · PSD/AI (작업 원본) |
| 색 공간 | sRGB |
| 최대 용량 | 5MB 이하 (유페이퍼 뷰어 호환성) |
| 텍스트 안전 영역 | 상하좌우 각 120px 여백 (썸네일 크롭 대비) |

> 📎 **팁** — 유페이퍼는 PDF/EPUB **첫 장에 표지가 포함** 되어야 등록됩니다. 업로드 전 PDF 파일을 열어 1페이지가 표지 이미지인지 반드시 확인하세요.

---

## 시각 컨셉 — 세 가지 방향

### A안 · "픽셀 → 핸들" 여정 (추천)

검정 배경 위에 **테슬라 차량의 전방 카메라 뷰가 픽셀 단위로 분해되어 스티어링 휠로 수렴** 하는 일러스트. 상단에 제목, 하단에 부제와 저자명. 픽셀 아트와 벡터 라인이 섞인 미래적 톤. 기술서답게 신뢰감을 주면서 책의 핵심 명제 *"Photon-in, Control-out"* 이 시각적으로 드러납니다.

**대표 색상** : `#0A0A0E` (딥 블랙) + `#E63946` (테슬라 레드 톤) + `#F1FAEE` (오프 화이트)
**타이포** : 제목은 *Pretendard Bold 72pt* / 부제 *Pretendard Regular 28pt*

### B안 · "같은 뇌, 다른 몸"

상단 절반은 **테슬라 차량의 측면 실루엣**, 하단 절반은 **Optimus 휴머노이드의 측면 실루엣**. 두 이미지가 공통의 신경망 그래프로 연결된 구성. 자율주행차와 휴머노이드가 같은 파운데이션 모델에서 출발한다는 이 책의 19장 핵심 메시지를 표지로 요약합니다.

**대표 색상** : `#0F172A` (네이비) + `#FBBF24` (앰버) + 화이트
**타이포** : 제목은 *Noto Sans KR Black 80pt*

### C안 · 미니멀 타이포그래피

일러스트 없이 **거대한 제목 타이포그래피만** 배치. *"테슬라처럼 만드는"* 을 작게, *"비전 자율주행과 피지컬 AI"* 를 크게, 그 아래 저자명과 시리즈명 *All That AI*. 오렌지 + 흰색 + 진회색 3색 미니멀. 기술서 중에서도 담담한 인상을 주는 스타일.

**대표 색상** : `#FF6B35` (오렌지) + `#FFFFFF` + `#2D3436`
**타이포** : 제목은 *Pretendard ExtraBold 96pt*

## 제작 방법 — 세 갈래

### 직접 제작 (무료)

- **Canva** (canva.com) 의 "전자책 표지" 템플릿에서 2:3 비율 선택 → 위의 A/B/C 안 중 하나를 본떠 텍스트 교체 → PNG 내보내기. 30분이면 완성 가능.
- **Figma** 로 세밀하게 작업 가능하지만 학습 곡선이 있음.

### 외주 의뢰

- **크몽 (kmong.com)** "전자책 표지 디자인" 카테고리. 기술서 경험 있는 디자이너에게 3~5만 원에 의뢰. 3~5일 소요.
- **부크크 디자인 외주** — 부크크 내부 디자이너 매칭 서비스. 5~15만 원.

### AI 생성 도구 (빠르게 초안이 필요할 때)

- **ChatGPT / Midjourney** 로 일러스트 생성 후 Canva 에서 텍스트 얹기. 저작권 주의. 상업 출판에는 외주가 안전합니다.

## 뒷표지 (POD 종이책 전환 시)

전자책 단독 출간에서는 필요 없지만, 부크크 POD 로 전환할 때를 위해 미리 준비해 두면 편합니다.

- 책 소개 요약 (300자 내외) — `책소개_상세.md` 의 "세 줄 소개" 를 확장
- 독자 추천 대상 (4~5줄)
- 저자 한 줄 프로필
- ISBN 바코드 (출판 시스템이 자동 삽입)
- 뒷면 하단 : *"본 도서의 실습 코드는 github.com/leelang7/physical-ai-book 에서 확인할 수 있습니다. QR 해설 영상 : youtube.com/@aidoer"*
