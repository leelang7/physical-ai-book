# 유페이퍼 실제 등록 절차 — Step-by-Step

> **소요 시간** : 가입부터 등록 완료까지 약 2시간. 검수는 영업일 기준 3~7일.
> **공식 URL** : [upaper.kr](https://upaper.kr/)
> **고객센터** : 공식 사이트 하단 "도움말" 참조

---

## 준비물 체크리스트

- [ ] **본문 + 표지 PDF** — 첫 페이지에 표지 이미지가 삽입된 단일 PDF (유페이퍼 필수 요건). 50MB 이하.
- [ ] **(선택) EPUB 파일** — EPUB 도 함께 제공하면 뷰어 경험이 개선되지만 PDF 만으로도 등록 가능.
- [ ] **표지 이미지 (별도)** — 가로 700~1,000px · 세로 1,000px PNG/JPG.
- [ ] **저자 정보** — 실명 · 연락처 · 계좌 · 사업자등록번호(있을 경우).
- [ ] **ISBN 신청 비용** — 1,000원 (등록 시 결제).

---

## 1단계 · PDF 에 표지 삽입

유페이퍼는 **원고 파일 첫 장에 표지가 반드시 포함** 되어야 합니다. Markdown Preview Enhanced 로 만든 PDF 에 표지를 앞장에 붙이는 가장 빠른 방법은 다음과 같습니다.

```
방법 A · PDF 병합 도구 (권장)
1. 표지 이미지를 A4 비율의 단일 페이지 PDF 로 변환
   - Canva 에서 PDF 다운로드 또는
   - Windows "인쇄 > Microsoft Print to PDF"
2. 무료 도구 ILovePDF 또는 Smallpdf 에서 Merge PDF 기능 이용
   - 1번 : 표지 PDF
   - 2번 : 본문 PDF
3. 결과물 파일명: 테슬라처럼만드는..._전자책_v1.0.pdf

방법 B · LaTeX / Pandoc 사용자
pandoc 명령어로 표지 이미지를 첫 페이지로 포함하는 옵션을 쓸 수 있지만,
방법 A 가 훨씬 빠르고 결과도 깔끔합니다.
```

---

## 2단계 · 유페이퍼 저자 가입

1. [upaper.kr](https://upaper.kr/) 접속 → 우측 상단 **저자 회원가입**
2. 필수 정보 입력 (실명 · 이메일 · 휴대폰 인증)
3. 정산 계좌 등록 (국내 은행 계좌)
4. 저자 프로필 작성 — `메타데이터.md` 의 "제공할 저자 정보" 블록을 그대로 복사

---

## 3단계 · 신간 등록 진입

1. 마이페이지 → **신간 등록** 메뉴
2. 전자책 유형 선택 : **일반 전자책 (PDF · EPUB)**
3. ISBN 신청 여부 : **예 (1,000원 결제)**

---

## 4단계 · 도서 상세 정보 입력

| 필드 | 입력값 |
|---|---|
| 제목 | 테슬라처럼 만드는 비전 자율주행과 피지컬 AI |
| 부제 | 카메라 한 대에서 Optimus까지, 픽셀이 핸들이 되는 여정 |
| 저자 | 이석창 |
| 출판사 | 이석창 (개인 출판) |
| 분야 | 컴퓨터/IT > 인공지능/머신러닝 |
| 도서 소개 | `책소개_상세.md` 의 한 줄 소개 + 세 줄 소개 + 이 책의 특별한 점 |
| 목차 | `책소개_상세.md` 의 "상세 목차" 복사 |
| 저자 소개 | `책소개_상세.md` 의 저자 소개 복사 |
| 키워드 | `메타데이터.md` 의 검색 키워드 15개 |
| 발행일 | 검수 통과 예상일로 설정 (보통 접수 + 7일) |

---

## 5단계 · 파일 업로드

1. **본문 파일** 업로드 : 1단계에서 만든 표지 포함 PDF (50MB 이하 확인)
2. **(선택) EPUB 파일** 업로드 : EPUB 이 있다면 함께 올려 뷰어 경험 개선
3. **표지 이미지(별도)** : 썸네일 용도로 PNG/JPG 별도 제출
4. 미리보기 확인

---

## 6단계 · 가격 설정 및 판매 채널 선택

### 가격

- 판매가 : **19,900원** (부크크와 동일)
- 런칭 할인 : 자사 프로모션 기능 이용해 4주간 14,900원 설정

### 판매 채널

- ☑ 유페이퍼 자사몰 (수수료 30% · 저자 70%)
- ☑ 교보문고 제휴 (수수료 40% · 저자 60%)
- ☑ 예스24 제휴 (수수료 40% · 저자 60%)
- ☑ 알라딘 제휴 (수수료 40% · 저자 60%)
- ☑ 기타 제휴 서점 (가능한 모두 선택)

제휴 채널을 모두 켜야 검색 유입이 극대화됩니다.

---

## 7단계 · 저작권 및 약관 동의 · 검수 제출

1. 저작권 보유 확인 체크 (본인 창작물임을 서약)
2. 유통 약관 동의
3. **검수 요청** 클릭
4. 검수 기간 : 3~7일
5. 이메일 · 마이페이지 알림으로 결과 수신

검수에서 자주 반려되는 이유 :
- PDF 첫 페이지에 표지가 없음
- 표지에 저작권 침해 이미지 사용
- 제목 · 저자명이 표지와 상세정보가 불일치

---

## 8단계 · 판매 개시 후 관리

- **저자 센터** 에서 채널별 판매량 실시간 확인
- 정산 : 매월 말일 기준 다음 달 15일경 입금 (최소 정산 금액 조건 있음)
- 도서 정보 수정은 마이페이지에서 직접 가능 (재검수 1~2일 소요)

---

## 부크크 · 유페이퍼 동시 등록 시 주의사항

- **파일명은 플랫폼별로 다르게** : 부크크용은 표지 없는 본문만, 유페이퍼용은 표지 포함 PDF 로 준비하는 것이 관리가 쉽습니다.
- **가격은 동일하게 유지** : 두 플랫폼 가격이 다르면 독자가 저렴한 쪽으로 몰려 한쪽의 노출이 떨어집니다.
- **판매 링크 관리** : 유튜브 · 블로그 · SNS 에 올릴 때는 부크크와 유페이퍼 링크를 **항상 나란히** 노출해 독자가 선택할 수 있게 하세요. 한쪽만 노출하면 반대 채널의 알고리즘 노출이 떨어집니다.
- **리뷰는 양쪽 모두 답글** : 두 플랫폼의 리뷰를 매주 한 번씩 확인해 감사 답글을 남기면, 신규 독자의 구매 전환이 눈에 띄게 올라갑니다.

## 자주 묻는 질문

**Q. ISBN 발급비 1,000원은 꼭 내야 하나요?**
A. ISBN 없이도 유페이퍼 자사몰 판매는 가능하지만, 교보·예스24·알라딘 제휴 판매에는 ISBN 이 필수입니다. 1,000원으로 유통 채널을 10배 이상 확장할 수 있으니 반드시 신청하세요.

**Q. EPUB 을 꼭 만들어야 하나요?**
A. 아닙니다. PDF 만으로도 등록·판매 가능합니다. 다만 EPUB 은 뷰어에서 글자 크기 조절이 가능해 독자 경험이 더 좋습니다. 여력이 되면 Calibre(무료 도구) 로 PDF → EPUB 변환을 시도해 보세요.

**Q. 부크크와 유페이퍼의 원고가 조금 달라져도 되나요?**
A. 가능합니다. 예를 들어 유페이퍼 버전에만 저자 후기 챕터를 추가하는 식의 차별화도 허용됩니다. 단, 제목이 다르면 두 권의 별개 도서로 간주될 수 있습니다.
