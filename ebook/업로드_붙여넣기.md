## 8. 세 줄 소개 (작품 설명 · 상단 본문)

```
국내 서점의 자율주행 교재 대부분은 여전히 LiDAR · HD Map 중심입니다. 이 책은 테슬라가 2024년 FSD v12 에서 보여 준 "픽셀이 곧 핸들이 되는" 세상을, Vision-only · End-to-End · Data Engine · Dojo · Optimus · VLA 로 이어지는 한 줄기의 이야기로 풀어냅니다. YOLO · MediaPipe 같은 친숙한 도구에서 출발해 MentorPi · ROS2 · Isaac Lab 까지, 책상 위의 작은 로봇이 실제로 움직이기 시작하는 순간까지 여러분을 안내합니다.
```

## 9. 이런 분께 추천 (작품 설명 · 중단 본문)

```
▶ 파이썬 기초는 뗐지만, 자율주행·Physical AI 로 커리어를 전환하고 싶은 3~8년차 엔지니어
▶ YOLO·MediaPipe 튜토리얼은 해봤지만 "그 다음" 이 막막한 대학원생과 취준생
▶ 자동차·로봇·드론 스타트업에서 Vision AI 를 도입해야 하는 현업 개발자
▶ AI 부트캠프 강사와 교재 제작자 — 한 학기 과정의 실습 교재로 바로 활용 가능
▶ 테슬라 FSD 의 원리가 실제로 어떻게 동작하는지 깊이 알고 싶은 엔지니어·연구자
```

## 10. 이 책의 특별한 점 (작품 설명 · 하단 본문)

```
하나. 국내 최초로 Tesla Vision-only × End-to-End × Physical AI 의 전체 스택을 한국어 중급 실무서로 꿰어낸 시도입니다. Tesla AI Day 자료, Wayve · Comma.ai 의 연구, 최근의 VLA 논문까지 한 줄기로 정리했습니다.

둘. "모방학습과 DAgger 를 주 무기로, 강화학습은 보조" 라는 분명한 학습 철학을 관통합니다. 저자가 2026년 발표한 논문 <See-ParkingNet: A Robust Imitation Learning Framework for Autonomous Mobile Robots via Geometric Perturbation of Synthetic Experts> 가 이 입장의 학술적 근거를 직접 제공합니다.

셋. 교재·영상·코드의 3중 동반 구조. YouTube 채널 "All That AI" (@aidoer) 와 GitHub 저장소 physical-ai-book 이 각 장과 1:1 로 연결됩니다.

넷. MentorPi 실증(29장) 과 ROS2 통합(30장) 으로 교실·연구실에서 실제로 로봇이 움직이는 경험까지 안내합니다.

다섯. 전 장이 서술형 산문으로 쓰였습니다. 개조식 나열과 딱딱한 교과서 문체를 배제하고, 수능 장문독해 · 카카오톡 이모티콘 · 플라이트 시뮬레이터 · 지하철 시운전 같은 친숙한 한국적 비유로 복잡한 개념의 입구를 열어 줍니다.
```
