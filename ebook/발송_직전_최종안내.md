## 🙋 저자님이 하실 5가지

### 1. PDF 육안 확인 (10분)
`ebook/전체원고_v2.0.pdf` 를 열어 처음·중간·끝 **3곳만** 훑어 보세요.
- 첫 페이지에 목차가 잘 생성됐는지
- 중간에 한글 줄바꿈이 이상한 곳 없는지
- 마지막 부록까지 빠짐없이 들어갔는지

이상하면 `scripts/build_pdf.py` 의 CSS 를 조정해 다시 빌드할 수 있습니다.

### 2. 부크크 업로드 (30분)
1. [bookk.co.kr](https://bookk.co.kr/) 로그인 → **책 만들기 → 전자책 만들기**
2. `ebook/업로드_붙여넣기.md` 의 1~14번 항목을 해당 입력란에 차례로 복붙
3. 원고 업로드 : `전체원고_v2.0.pdf`
4. 표지 업로드 : `이석창_테슬라처럼만드는비전자율주행과피지컬AI_cover.png`
5. ISBN 자동 발급 체크 → 검수 요청

### 3. 유페이퍼 업로드 (40분)
1. [upaper.kr](https://upaper.kr/) 가입 → **마이페이지 → 신간 등록**
2. 같은 메타데이터 재사용 (1~14번 그대로)
3. 원고 업로드 : `전체원고_v2.0_유페이퍼.pdf` (표지 포함 버전)
4. 표지 별도 업로드 : 같은 PNG
5. 판매 채널 모두 체크 (교보·예스24·알라딘 제휴)
6. ISBN 1,000원 결제 → 검수 요청

### 4. GitHub 저장소 About 설정 (5분)
https://github.com/leelang7/physical-ai-book
우측 상단 ⚙️ (About 편집) 눌러서 :
- **Description** : "테슬라처럼 만드는 비전 자율주행과 피지컬 AI · All That AI Vol.01 · 이석창 지음"
- **Website** : https://www.youtube.com/@aidoer
- **Topics** : `autonomous-driving` `tesla` `physical-ai` `imitation-learning` `dagger` `ros2` `korean-book` `vla` `optimus` `mentorpi`

### 5. 검수 승인 후 런칭 (즉시)
검수 승인 이메일을 받으면 **같은 날** `ebook/런칭_템플릿.md` 를 열어서 :
- GitHub Release v1.0 태그 생성 (Release notes 는 템플릿 그대로)
- YouTube 런칭 영상 녹화·업로드
- 블로그·Threads·LinkedIn 공지
- 수강생 이메일 순차 발송

## 📞 문제 발생 시 빠른 대응

| 증상 | 대응 |
|---|---|
| 부크크 검수 반려 "표지 해상도 부족" | `cover/cover_A_red_column.png` 재업로드 (1600×2400 · 300dpi 명시) |
