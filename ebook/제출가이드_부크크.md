# 부크크 실제 등록 절차 — Step-by-Step

> **소요 시간** : 가입부터 등록 완료까지 약 2~3시간. 검수는 영업일 기준 3~5일.
> **공식 URL** : [bookk.co.kr](https://bookk.co.kr/)
> **고객센터** : 1544-0252

---

## 준비물 체크리스트

- [ ] **본문 원고 PDF** — `전체원고_v1.0.md` 를 PDF 로 변환한 파일. 권장: A4 · 10~11pt · 1.5 줄간격.
- [ ] **표지 이미지** — 1,600 × 2,400 px 2:3 비율 PNG. `표지_컨셉.md` 참조.
- [ ] **저자 정보** — 실명 · 주민등록번호(인세 정산 세금계산용) · 입금 계좌.
- [ ] **책 소개 원고** — `책소개_상세.md` 의 세 줄 소개, 상세 설명, 저자 소개.
- [ ] **메타데이터** — `메타데이터.md` 의 카테고리 · 키워드 · 저자 프로필.
- [ ] **ISBN 신청 정보** — 부크크가 자동 부여하므로 별도 제출 불필요.

---

## 1단계 · MD → PDF 변환 (VSCode Markdown Preview Enhanced)

```
1. VSCode 에서 c:/lsc/Tesla_book/ebook/전체원고_v1.0.md 열기
2. 우클릭 → Markdown Preview Enhanced 미리보기
3. 미리보기 창에서 우클릭 → Chrome (Puppeteer) > PDF
4. 저장 위치: c:/lsc/Tesla_book/ebook/
5. 파일명: 테슬라처럼만드는비전자율주행과피지컬AI_v1.0.pdf
```

PDF 변환 직후 반드시 열어서 **첫 페이지에 표지가 삽입되어 있는지** 확인하세요. 부크크는 본문과 표지를 별도 업로드하지만, 유페이퍼는 표지가 포함되어야 하므로 같은 PDF 를 양쪽에 쓰려면 첫 장 표지 삽입이 편리합니다.

### 페이지 세팅 추천 (MPE config)

마크다운 파일 최상단에 다음 Front Matter 를 추가하면 PDF 가 더 깔끔해집니다.

```yaml
---
title: "테슬라처럼 만드는 비전 자율주행과 피지컬 AI"
author: "이석창"
print_background: true
---
```

## 2단계 · 부크크 가입 및 신간 등록 진입

1. [bookk.co.kr](https://bookk.co.kr/) 접속 → 상단 **회원가입**
2. 이메일 · 실명 · 연락처 입력 후 인증
3. 메인 상단의 **책 만들기** → **전자책 만들기** 클릭
4. "신간 전자책 등록" 시작

---

## 3단계 · 도서 정보 입력

| 필드 | 입력값 |
|---|---|
| 도서 제목 | 테슬라처럼 만드는 비전 자율주행과 피지컬 AI |
| 부제 | 카메라 한 대에서 Optimus까지, 픽셀이 핸들이 되는 여정 |
| 저자명 | 이석창 |
| 책 소개 | `책소개_상세.md` 의 "세 줄 소개" + "이런 분께 추천합니다" 복사 |
| 상세 설명 | `책소개_상세.md` 의 "이 책의 특별한 점" + "상세 목차" 복사 |
| 저자 소개 | `메타데이터.md` 의 "제공할 저자 정보" 블록 복사 |
| 카테고리 | IT · 컴퓨터 > 인공지능 · 머신러닝 > 컴퓨터 비전 · 자율주행 |
| 키워드 | `메타데이터.md` 의 검색 키워드 15개 복사 |

## 4단계 · 원고 및 표지 업로드

1. **원고 파일** 탭 → PDF 선택 → 1단계에서 만든 `테슬라처럼만드는..._v1.0.pdf` 업로드
2. **표지 이미지** 탭 → PNG 선택 → 1,600 × 2,400 px 표지 업로드
3. 미리보기 확인 — 표지 · 본문이 정상 표시되는지 체크

---

## 5단계 · 가격 설정

- 전자책 판매가 : **19,900원** (런칭 직후 14,900원으로 낮출 예정)
- 할인 프로모션 : "런칭 기념" 4주 한정 25% 할인 설정
- 인세율 : 기본값 유지 (부크크 자사 판매 기준 50% 내외 자동 적용)

---

## 6단계 · ISBN 및 검수 제출

1. **ISBN 자동 발급** 체크
2. **검수 요청** 버튼 클릭
3. 부크크 검수팀이 본문 · 표지 · 메타데이터 확인 (영업일 3~5일)
4. 이메일로 검수 결과 수신 → 승인 시 자동으로 판매 개시

검수에서 반려되는 흔한 이유 :
- 표지 해상도 부족 (1,500px 이하)
- 본문 PDF 의 글자 깨짐 (폰트 임베드 누락)
- 저작권 문구 누락

대부분은 수정 후 재제출로 해결됩니다.

---

## 7단계 · 판매 개시 후 관리

- **저자 센터** 대시보드에서 실시간 판매 수량 확인
- 인세 정산 : 매월 말일 기준, 다음 달 15일경 입금 (보통 최소 인세 조건 충족 후)
- 독자 리뷰 관리 : 상세페이지 리뷰에 성실히 답글

## 자주 묻는 질문

**Q. 같은 원고를 유페이퍼에도 올릴 수 있나요?**
A. 네, 부크크는 기본 비독점 계약입니다. 다만 저작권 신고 화면에서 "독점 출간" 에 체크하지 않도록 주의하세요.

**Q. 실습 코드 링크나 유튜브 QR 은 본문 어디에 넣나요?**
A. 각 장 끝의 "더 읽을 거리" 섹션에 이미 QR 코드 자리가 마련되어 있습니다. PDF 생성 전에 각 QR 위치에 실제 짧은 URL(bit.ly 단축 URL) 을 삽입하면 됩니다. 출간 후에도 같은 URL 을 유지하도록 리다이렉트 설정을 걸어두세요.

**Q. 판매 가격을 나중에 바꿀 수 있나요?**
A. 저자 센터에서 가격 변경 신청 → 검수 재통과(1~2일) 후 반영됩니다. 할인 프로모션은 즉시 반영됩니다.
