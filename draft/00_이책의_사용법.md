# 이 책의 사용법 · 학습 경로 지도

## 이 책을 읽는 세 가지 방법

### 경로 ① 순서대로 완독 (권장)
1장부터 28장까지 차례로. 전체 약 **3~4개월**(매일 30분 + 주말 실습 2시간).
가장 효과적이지만 시간이 많이 듭니다.

### 경로 ② 실무 최소 경로 (빠른 진입)
취업·이직이 급할 때.
- 1~3장 (배경 이해)
- 5~8장 (Perception 핵심)
- 10장 (Planner)
- 23, 24장 (환경 + 미니 HydraNet)
- 27장 (Physical AI 실습)
→ 약 **4~6주**

### 경로 ③ Physical AI 집중
로보틱스·휴머노이드 방향.
- 1, 3장 (패러다임 이해)
- 18~22장 (Physical AI 전체)
- 20, 21, 27장 (실습)
- 부록 B의 로봇학습 논문
→ 약 **6~8주**

## 실습 환경 체크리스트

- [ ] Python 3.11 + PyTorch 2.5+
- [ ] Docker (선택, 재현성)
