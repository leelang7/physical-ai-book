# 정오표 (Errata)

도서 *"테슬라처럼 만드는 비전 자율주행과 피지컬 AI"* 의 정정 사항이 누적되는 문서입니다. 독자분께서 [Issues](https://github.com/leelang7/physical-ai-book/issues) 에 제보해 주신 내용을 검토 후 이 파일에 반영합니다.

## 사용법

각 항목은 다음 형식으로 적습니다.

```
### v1.0 · 2026-04-27 출간
- p25, 1장 본문 — "원문 잘못된 부분" → "정정된 부분" (제보: @username, 반영일: 2026-MM-DD)
```

페이지 번호는 부크크 PDF 기준이며, 유페이퍼 PDF 의 경우 +1 (표지 페이지 포함) 입니다.

---

## v1.0 (2026-04-27 초판)

> 현재까지 보고된 정정 사항이 없습니다.

---

## 제보 방법

1. [Issues 새로 열기](https://github.com/leelang7/physical-ai-book/issues/new)
2. 제목에 `[정오] 페이지번호 — 짧은 설명` 형식으로 작성
3. 본문에 (a) 현재 표현 (b) 제안 표현 (c) 그렇게 보는 이유를 적어 주세요

확인 후 1주일 안에 답글 드리며, 다음 쇄·개정판에 반영합니다. 감사합니다.
